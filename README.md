# Zombie API Discovery and Defence Platform

## Overview
A platform for continuous discovery, classification, and remediation of undocumented, deprecated, or unused ("Zombie") APIs across distributed banking infrastructure.

## Team Roles
- **Lead/Author:** Repository management, workflow setup
- **Backend / API Discovery Engineer:** Automated API discovery, inventory, usage tracking
- **Security & Risk Analysis Engineer:** API classification, security assessment, risk scoring
- **Frontend & DevOps Engineer:** Dashboard, alerting, decommissioning workflows, deployment

## Features
- Automated API discovery (code, network, traffic)
- API inventory and lifecycle classification
- Security posture assessment
- Remediation and decommissioning workflows
- Continuous monitoring and alerting
